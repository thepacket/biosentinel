# Contributing

**Biosentinel** is an independent, personal project: a studio for designing
whole-cell CRISPR biosensors in safe (BSL-1 / GRAS) bacteria, maintained by
[@thepacket](https://github.com/thepacket).

## Pull requests are not accepted

This project does **not** accept pull requests — any PR opened here is closed
automatically. This is a deliberate choice to keep the project's provenance and
direction clear. You are very welcome to **fork** and build your own version.

## Issues

Bug reports, ideas, and feedback are genuinely welcome via the **Issues** tracker.

## Safety note

All chassis in this project are non-pathogenic **BSL-1 / GRAS** organisms, and
the schema enforces `biosafetyLevel == 1`. Designs marked *template* combine
individually-validated parts into a CRISPR circuit pattern and are **not**
end-to-end validated devices. This repository is for **research / educational
use only** — engineer responsibly and follow your institution's biosafety rules.
